[Css] CSS3 TIP

jeonyoungho Sep 3, 2021 2021-09-03T00:00:00+09:00
5 min

2021л…„ 8мӣ” 31мқј

1) мҡ”мҶҢ м„ёлЎң к°ҖмҡҙлҚ° м •л ¬н• л•Ң л¶ҖлӘЁ мҡ”мҶҢмқҳ displayлҘј flexлЎң м„ м–ён•ҳкі  align-items: center; лҘј м¶”к°Җн•ҳл©ҙ лҗңлӢӨ.

2) мҡ”мҶҢ м„ёлЎң к°ҖмҡҙлҚ° м •л ¬н• л•Ң л¶ҖлӘЁ мҡ”мҶҢмқҳ displayлҘј flexлЎң м„ м–ён•ҳкі  justify-content: center; лҘј м¶”к°Җн•ҳл©ҙ лҗңлӢӨ.

3) widthмҡ”мҶҢмқҳ нҚјм„јнӢ°м§Җ л№„мңЁмқ„ мӨ„л•Ң л¶ҖлӘЁ мҡ”мҶҢмқҳ кё°мӨҖмңјлЎң н•ҙлӢ№ нҚјм„јнӢ°м§Җл§ҢнҒј кі„мӮ°лҗңлӢӨ.

2021л…„ 9мӣ” 1мқј

1) line-height мҳҒн–Ҙ л°ӣлҠ” кІғ: мқёлқјмқёмҡ”мҶҢ, мқёлқјмқёлё”лЎқ мҡ”мҶҢ(лё”лЎқмҡ”мҶҢлҠ” line-heightлЎң мҲҳм§Ғ м •л ¬н•  мҲҳ м—ҶмқҢ)

2) м •м„қм Ғмқё м„ёлЎң м •л ¬ л°©лІ•:в‘  line-heightлҘј мқҙмҡ©н•ң л°©лІ• в‘Ў л¶ҖлӘЁ мҡ”мҶҢмқҳ paddingмқ„ мң„ м•„лһҳлЎң мЈјлҠ” л°©лІ•

3) !importantлҠ” мөңлҢҖн•ң м•ҲмҚЁмЈјлҠ”кІҢ мўӢмқҢ

4) л¶ҖлӘЁк°Җ flex м»Ён…Ңмқҙл„Ҳмқё мғҒнҷ©м—җм„ң мһҗмӢқн•ңн…Ң margin-[direction]: autoлҘј мЈјкІҢлҗҳл©ҙ лӮЁмқҖ кіөк°„мқ„ marginмқҙ м°Ём§Җн•ҳкІҢ лҗңлӢӨ.

5) css нҢҢмқј мһ‘м„ұ мӢң мҶҚм„ұ мҲңм„ңлҘј л§һм¶°мЈјлҠ”кІҢ мўӢлӢӨ.(л°°м№ҳ л°Ҹ мң„м№ҳ вҶ’ нҳ„мһ¬ мҡ”мҶҢмқҳ мҶҚм„ұ)

2021л…„ 9мӣ” 2мқј

1) мҡ”мҶҢ мҲҳнҸү м •л ¬н•ҳлҠ” мү¬мҡҙ л°©лІ•

  • мһҗмӢқ мҡ”мҶҢм—җ margin: 0 auto;лҘј м„Өм •н•ҳл©ҙ мўҢмҡ° л§Ҳм§„мқҙ лҸҷмқјн•ҳкІҢ м„Өм •лҗҳм–ҙ мҲҳнҸү м •л ¬лҗңлӢӨ.

2) domм—ҳлҰ¬лЁјнҠёмқҳ class лҳҗлҠ” id м§Җм •мӢң лӢӨлҘё нҺҳмқҙм§ҖлҘј нҸ¬н•Ён•ҳм—¬ мӨ‘ліөлҗҳм–ҙ мӮ¬мҡ©н•  мҲҳ мһҲлҠ” мҶҚм„ұмқҳ кІҪмҡ° classлЎң м„ м–ён•ҳкі  н•ң лІҲл§Ң мӮ¬мҡ©лҗ  кІҪмҡ° idлЎң м§Җм •н•ңлӢӨ.

2021л…„ 9мӣ” 3мқј

1) vertical-align: middle

  • мқёлқјмқё мҡ”мҶҢл“ӨлҒјлҰ¬ мҲҳм§Ғ м •л ¬мӢң (м»Ён…Ңмқҙл„Ҳм—җ мЈјлҠ” мҶҚм„ұмқҙ м•„лӢҢ мһҗмӢқ мҡ”мҶҢм—җ мЈјлҠ” мҶҚм„ұ)
  • н…Ңмқҙлё” м…Җмқјл•Ң мҲҳм§Ғ м •л ¬мӢң

2) мқҙлҜём§ҖлҘј ліј л•Ң нҷ•лҢҖн•ҙм„ң мһҗм„ён•ҳкІҢ ліј кІғ

  • мЈјм–ҙм§„ мқҙлҜём§Җ нҢҢмқјмқ„ мһҗм„ёнһҲ нҷ•лҢҖн•ҙм„ң л“Өм—¬лӢӨлҙ„мңјлЎңмҚЁ box-shadowк°Җ мһҲлҠ”м§Җ, к·ёлқјлҚ°мқҙм…ҳмқҙ м Ғмҡ©лҗҳм—ҲлҠ”м§ҖлҘј мң мқҳн•  кІғ

3) мҷ л§Ңн•ҳл©ҙ л¶ҖлӘЁ мҡ”мҶҢ, мһҗмӢқ мҡ”мҶҢ лӘЁл‘җ мқјмқјмқҙ pxлЎң нҒ¬кё°лҘј м§Җм •н•ҳм§Җ л§җ кІғ! (л¬ҙмЎ°кұҙ mustлҠ” м•„лӢҳ!)

  • мҷң к·ёлһҳм•ј н•ҳлӮҳ? -> ліҖнҷ”м—җ мң м—°н•ң к°ңл°ңмқ„ н•ҳкё° мң„н•ҙм„ң!
  • л¶ҖлӘЁмҡ”мҶҢлҘј 100pxлЎң м§Җм •н•ҳкі  мһҗмӢқ мҡ”мҶҢ 2к°ңлҘј 50px + 50pxлЎң мқјмқјмқҙ м§Җм •н•  мҲҳ мһҲм§Җл§Ң л§Ңм•Ҫ мһҗмӢқ мҡ”мҶҢк°Җ н•ҳлӮҳ лҚ” м¶”к°Җ лҗңлӢӨл©ҙ л¶ҖлӘЁмҡ”мҶҢмқҳ нҒ¬кё°лҸ„ мҲҳм •н•ҙмӨҳм•ј лҗ  кІғмқҙлӢӨ! мқҙлҠ” мң м§ҖліҙмҲҳм Ғмқё мёЎл©ҙм—җм„ң мўӢм§Җ м•ҠлӢӨ!
  • л”°лқјм„ң мһҗмӢқ мҡ”мҶҢ 2к°ңлҘј 50px + 50pxлЎң м§Җм •н•ҙ лҶ“кі  л¶ҖлӘЁмҡ”мҶҢлҠ” мһҗм—°мҠӨл Ҳ 100pxмқҙ лҗҳлҠ” нқҗлҰ„мңјлЎң м Ғмҡ©н•ҙм•ј н•ңлӢӨ.
  • н•ҳм§Җл§Ң л¬ҙмЎ°кұҙмқҖ м•„лӢҲлӢӨ. мҠӨнҒ¬лЎӨл°”мҷҖ к°ҷмқҖ мҳҲмҷёмӮ¬н•ӯмқҖ мЎҙмһ¬н•ңлӢӨ!

4) мңҲлҸ„мҡ°мқҳ кё°ліё мқҙлҜём§Җ л·°м–ҙлҠ” мӢӨм ң мқҙлҜём§ҖлҘј нҷ•лҢҖн•ҙм„ң мһҗм„ён•ҳкІҢ ліҙкё°м—” л¬ҙлҰ¬к°Җ мһҲлӢӨ. л”°лқјм„ң кҝҖл·°мқҳ н•„н„° м Ғмҡ© м•Ҳн•Ё мҳөм…ҳмқ„ м Ғмҡ©н•ҳм—¬ нҷ•лҢҖ нӣ„ м°ёкі н•ҳлҠ” кІғмқҙ мўӢлӢӨ.

лҚ” мһҗм„ён•ң лӮҙмҡ©мқҖ м—¬кё°лҘј м°ёкі н•ҳмһҗ.

мқјл°ҳм ҒмңјлЎң мӨ‘мҡ”н•ң кІғл¶Җн„° лҚң мӨ‘мҡ”н•ңкІғмқҳ мҲңм„ңлЎң лӮҳм—ҙн•ҳлҠ”кІҢ ліҙнҺём ҒмңјлЎң мӮ¬мҡ©лҗңлӢӨ. мҳҲлҘј л“Өм–ҙ, л°°м№ҳ -> мң„м№ҳ -> нҳ„мһ¬ мҡ”мҶҢмқҳ мҶҚм„ұ(block/inline) вҖҰ
