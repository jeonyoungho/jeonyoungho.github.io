[Java] WeakHashMap

jeonyoungho May 23 2022-05-23T00:00:00+09:00
5 min

Javaм—җм„ңлҠ” м„ё к°Җм§Җ мЈјмҡ” мң нҳ•мқҳ м°ёмЎ°(Reference) л°©мӢқ

к°•н•ң м°ёмЎ° (Strong Reference)

  • Integer prime = 1; мҷҖ к°ҷмқҖ к°ҖмһҘ мқјл°ҳм Ғмқё м°ёмЎ° мң нҳ•мқҙлӢӨ.
  • prime ліҖмҲҳ лҠ” к°’мқҙ 1 мқё Integer к°қмІҙм—җ лҢҖн•ң к°•н•ң м°ёмЎ°лҘј к°Җм§„лӢӨ.
  • мқҙ к°қмІҙлҘј к°ҖлҰ¬нӮӨлҠ” к°•н•ң м°ёмЎ°к°Җ мһҲлҠ” к°қмІҙлҠ” GCлҢҖмғҒмқҙ лҗҳм§Җм•ҠлҠ”лӢӨ.

л¶Җл“ңлҹ¬мҡҙ м°ёмЎ° (Soft Reference)

  • SoftReference<Integer> soft = new SoftReference<Integer>(prime); мҷҖ к°ҷмқҙ SoftReference ClassлҘј мқҙмҡ©н•ҳм—¬ мғқм„ұмқҙ к°ҖлҠҘн•ҳлӢӨ.
  • л§Ңм•Ҫ prime == null мғҒнғңк°Җ лҗҳм–ҙ лҚ”мқҙмғҒ мӣҗліё(мөңмҙҲ мғқм„ұ мӢңм җм—җ мқҙмҡ© лҢҖмғҒмқҙ лҗҳм—ҲлҚҳ Strong Reference) мқҖ м—Ҷкі  лҢҖмғҒмқ„ м°ёмЎ°н•ҳлҠ” к°қмІҙк°Җ SoftReferenceл§Ң мЎҙмһ¬н•  кІҪмҡ° GCлҢҖмғҒмңјлЎң л“Өм–ҙк°ҖлҸ„лЎқ JVMмқҖ лҸҷмһ‘н•ңлӢӨ.
  • лӢӨл§Ң WeakReference мҷҖмқҳ м°Ёмқҙм җмқҖ л©”лӘЁлҰ¬к°Җ л¶ҖмЎұн•ҳм§Җ м•Ҡмңјл©ҙ көімқҙ GCн•ҳм§Җ м•ҠлҠ” м җмқҙлӢӨ.
  • л•Ңл¬ём—җ мЎ°кёҲмқҖ м—„кІ©н•ҳм§Җ м•ҠмқҖ Cache Libraryл“Өм—җм„ң л„җлҰ¬ мӮ¬мҡ©лҗҳлҠ” кІғмңјлЎң м•Ңл Өм ёмһҲлӢӨ.

м•Ҫн•ң м°ёмЎ° (Weak Reference)

  • WeakReference<Integer> soft = new WeakReference<Integer>(prime); мҷҖ к°ҷмқҙ WeakReference ClassлҘј мқҙмҡ©н•ҳм—¬ мғқм„ұмқҙ к°ҖлҠҘн•ҳлӢӨ.
  • prime == null лҗҳл©ҙ (н•ҙлӢ№ к°қмІҙлҘј к°ҖлҰ¬нӮӨлҠ” м°ёмЎ°к°Җ WeakReference лҝҗмқј кІҪмҡ°) GC лҢҖмғҒмқҙ лҗңлӢӨ.
  • м•һм„ң мқҙм•јкё° н•ң лӮҙмҡ©кіј к°ҷмқҙ SoftReferenceмҷҖ м°Ёмқҙм җмқҖ л©”лӘЁлҰ¬к°Җ л¶ҖмЎұн•ҳм§Җ м•ҠлҚ”лқјлҸ„ GC лҢҖмғҒмқҙ лҗңлӢӨлҠ” кІғмқҙлӢӨ.
  • к·ёлҰ¬кі  лӮҳм„ң лӢӨмқҢ GCк°Җ л°ңмғқн•ҳлҠ” мӢңм җм—җ л¬ҙмЎ°кұҙ м—Ҷм–ҙм§„лӢӨ.

WeakHashMap

  • мқјл°ҳм Ғмқё HashMapмқҳ кІҪмҡ° мқјлӢЁ Mapм•Ҳм—җ KeyмҷҖ Valueк°Җ putлҗҳл©ҙ мӮ¬мҡ©м—¬л¶ҖмҷҖ кҙҖкі„м—Ҷмқҙ н•ҙлӢ№ лӮҙмҡ©мқҖ мӮӯм ңлҗҳм§Җ м•ҠлҠ”лӢӨ.
  • Mapм•Ҳмқҳ Elementл“Өмқҙ мқјл¶ҖлҠ” мӮ¬мҡ©лҗҳкі  мқјл¶ҖлҠ” мӮ¬мҡ©лҗҳм§Җ м•Ҡмқ„ мҲҳ мһҲлҠ” кІҪмҡ°лҸ„ мһҲмңјлӮҳ, к·ёкІғмқҳ кө¬нҳ„мқҖ м „м ҒмңјлЎң н”„лЎңк·ёлһҳлЁём—җ лӢ¬л ӨмһҲкІҢ лҗңлӢӨ.
  • мҳҲлҘј л“Өл©ҙ Keyм—җ н•ҙлӢ№н•ҳлҠ” к°қмІҙк°Җ лҚ”мқҙмғҒ мЎҙмһ¬н•ҳм§Җ м•ҠкІҢ лҗҳлҠ” кІҪмҡ°мқҙлӢӨ. к·ёлһҳм„ң л§Ңм•Ҫ м–ҙл–Ө к°қмІҙк°Җ nullмқҙ лҗҳм–ҙ лІ„лҰ¬л©ҙ н•ҙлӢ№ к°қмІҙлҘј keyлЎң н•ҳлҠ” HashMapмқҳ ElementлҸ„ лҚ”мқҙмғҒ кәјлӮј мқјмқҙ м—ҶлҠ” кІҪмҡ°к°Җ л°ңмғқн•ҳлҠ” кІғмқ„ к°Җм •н•ҙліј мҲҳ мһҲлӢӨ.
  • WeakHashMapмқҖ WeakReferenceмқҳ нҠ№м„ұмқ„ мқҙмҡ©н•ҳм—¬ HashMapмқҳ ElementлҘј мһҗлҸҷмңјлЎң м ңкұ°, GC н•ҙлІ„лҰ°лӢӨ. Keyм—җ н•ҙлӢ№н•ҳлҠ” к°қмІҙк°Җ лҚ”мқҙмғҒ мӮ¬мҡ©лҗҳм§Җ м•ҠлҠ”лӢӨкі  нҢҗлӢЁлҗҳл©ҙ м ңкұ°н•ңлӢӨлҠ” мқҳлҜёмқҙлӢӨ. м•„лһҳмқҳ мӮ¬мҡ©мҳҲлҘј ліҙмһҗ.

public class WeakHashMapTest {
    public static void main(String[] args) {
        WeakHashMap<Integer, String> map = new WeakHashMap<>();
        Integer key1 = 1000;
        Integer key2 = 2000;

        map.put(key1, "test a");
        map.put(key2, "test b");

        key1 = null;

        System.gc();  //к°•м ң Garbage Collection
        map.entrySet().stream().forEach(el -> System.out.println(el));
    }
}

  • мқҙ нҒҙлһҳмҠӨлҠ” мЈјлЎң equals л©”мҶҢл“ңк°Җ == м—°мӮ°мһҗлҘј мӮ¬мҡ©н•ҳлҠ” KeyлҘј мӮ¬мҡ©н•  кІҪмҡ° мң мҡ©н•ҳлӢӨ.
    • мҰү к°қмІҙмқҳ мЈјмҶҢк°’ л№„көҗлҘј н•ҳлҠ” кІҪмҡ°лқј ліј мҲҳ мһҲлӢӨ.
  • к·ёлҹ° Keyк°Җ лІ„л Өм§Җл©ҙ кІ°мҪ” лҸҷмқјн•ң KeyлҠ” мғқм„ұлҗҳм§Җ м•ҠмңјлҜҖлЎң, лӮҳмӨ‘м—җ WeakHashMap м—җм„ң н•ҙлӢ№ нӮӨмқҳ мЎ°нҡҢлҘј мҲҳн–үн•ҳлҠ” кІғмқҖ л¶Ҳк°ҖлҠҘн•ҳкІҢ лҗҳкё° л•Ңл¬ёмқҙлӢӨ.
  • к·ёлҹ¬лӮҳ Keyк°Җ String кіј к°ҷмқҙ лӢЁмқј к°’мқ„ к°Җм§ҖлҠ” Classкі  == м—°мӮ°мһҗ мҷём—җ equalsлҘј мӮ¬мҡ©н•ҳм—¬ м„ңлЎң лӢӨлҘё к°қмІҙлқјлҸ„ лҸҷмқјм„ұмқ„ мІҙнҒ¬н•ҳм—¬ мң м§Җ н•ҳкі  мӢ¶мқҖ кІҪмҡ° м Ғм Ҳн•ҳм§Җ м•ҠлӢӨ.
    • мЈјмҶҢк°’мқ„ л№„көҗн•ҳлҠ”кІҢ м•„лӢҢ к°қмІҙмқҳ мӢӨм ң лӮҙл¶ҖлҘј л№„көҗн•ҳлҠ” кІҪмҡ°лқј ліј мҲҳ мһҲлӢӨ.
  • WeakHashMapмқҖ HashMapкіј л§Ҳм°¬к°Җм§ҖлЎң Thread-safeн•ҳм§Җ м•Ҡмңјл©° н•„мҡ”н•  кІҪмҡ° Collections.synchronizedMap мқ„ мқҙмҡ©н•  мҲҳ мһҲлӢӨ.

Note: StringмқҖ newк°Җ м•„лӢҢ лҰ¬н„°лҹҙ л°©мӢқмңјлЎң мғқм„ұлҗ  кІҪмҡ° (String a = вҖңabcвҖқ; мҷҖ к°ҷмқҙ) intern()м—җ мқҳн•ҙ String poolмқ„ JVMмқҖ мӮ¬мҡ©н•ҳлҜҖлЎң keyк°Җ nullмқҙ лҗңлӢӨкі  н•ҳлҸ„ WeakHashMapмқҖ мһҗлҸҷмңјлЎң мӮӯм ңлҗҳм§Җ м•ҠлҠ”лӢӨ. мқҙлҠ” Integer мҷҖ к°ҷмқҖ Classмқҳ (-127 ~128) к°’л“ӨлҸ„ л§Ҳм°¬к°Җм§Җ мқҙлӢӨ. JVMмқҖ лҜёлҰ¬ л¶ҲліҖ к°қмІҙлЎң н•ҙлӢ№ к°’мқ„ ліҙкҙҖн•ҳкі  мһҲлӢӨ. (Strong м°ёмЎ°)
