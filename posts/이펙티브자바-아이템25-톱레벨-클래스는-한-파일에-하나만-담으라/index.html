  • н•ҳм§Җл§Ң мӢ¬к°Ғн•ң мң„н—ҳмқ„ к°җмҲҳн•ҙм•ј н•ҳлҠ” н–үмң„лӢӨ.
    • мқҙл ҮкІҢ н•ҳл©ҙ н•ң нҒҙлһҳмҠӨлҘј м—¬лҹ¬ к°Җм§ҖлЎң м •мқҳн•  мҲҳ мһҮмңјл©°, к·ёмӨ‘ м–ҙлҠҗ кІғмқ„ мӮ¬мҡ©н• м§ҖлҠ” м–ҙлҠҗ мҶҢмҠӨ нҢҢмқјмқ„ лЁјм Җ м»ҙнҢҢмқјн•ҳлҠҗлғҗм—җ л”°лқј лӢ¬лқјм§Җкё° л•Ңл¬ёмқҙлӢӨ.

// мҪ”л“ң 25-1 л‘җ нҒҙлһҳмҠӨк°Җ н•ң нҢҢмқј(Utensil.java)м—җ м •мқҳлҗҳм—ҲлӢӨ. - л”°лқј н•ҳм§Җ л§җ кІғ! (150мӘҪ)
class Utensil {
    static final String NAME = "pan";
}

class Dessert {
    static final String NAME = "cake";
}
  • мҡҙ мўӢкІҢ javac Main.java Dessert.java лӘ…л №мңјлЎң м»ҙнҢЎлҰ¬н•ңлӢӨл©ҙ м»ҙнҢҢмқј мҳӨлҘҳк°Җ лӮҳкі  Utensilкіј Dessert нҒҙлһҳмҠӨлҘј мӨ‘ліө м •мқҳн–ҲлӢӨкі  м•Ңл ӨмӨ„ кІғмқҙлӢӨ.
  • м»ҙнҢҢмқјлҹ¬лҠ” к°ҖмһҘ лЁјм Җ Main.javaлҘј м»ҙнҢҢмқј н•ҳкі , к·ём•Ҳм—җм„ң (Dessertм°ёмЎ°ліҙлӢӨ лЁјм Җ лӮҳмҳӨлҠ”) Utensil м°ёмЎ°лҘј л§ҢлӮҳл©ҙ Utensil.java нҢҢмқјмқ„ мӮҙнҺҙ Utensilкіј DessertлҘј лӘЁл‘җ м°ҫм•„лӮј кІғмқҙлӢӨ.
  • к·ёлҹ° лӢӨмқҢ м»ҙнҢҢмқјлҹ¬к°Җ л‘җ лІҲм§ё лӘ…л №мӨ„ мқёмҲҳлЎң л„ҳм–ҙмҳЁ Dessert.javaлҘј мІҳлҰ¬н•ҳл Ө н•  л•Ң к°ҷмқҖ нҒҙлһҳмҠӨмқҳ м •мқҳк°Җ мқҙлҜё мһҲмқҢмқ„ м•ҢкІҢ лҗңлӢӨ.
  • мқҙмІҳлҹј м»ҙнҢҢмқјлҹ¬м—җ м–ҙлҠҗ мҶҢмҠӨ нҢҢмқјмқ„ лЁјм Җ кұҙл„ӨлҠҗлғҗм—җ л”°лқј лҸҷмһ‘мқҙ лӢ¬лқјм§ҖлҜҖлЎң л°ҳл“ңмӢң л°”лЎң мһЎм•„м•јн•  л¬ём ңмқҙлӢӨ.

н•ҙкІ°мұ…

  • н•ҙкІ°мұ…мқҖ м•„мЈј к°„лӢЁн•ҳлӢӨ. нҶұл ҲлІЁ нҒҙлһҳмҠӨл“Ө(Utensilкіј Dessert)лҘј м„ңлЎң лӢӨлҘё мҶҢмҠӨ нҢҢмқјлЎң л¶„лҰ¬н•ҳл©ҙ лҗңлӢӨ.
  • көімқҙ м—¬лҹ¬ нҶұл ҲлІЁ нҒҙлһҳмҠӨлҘј н•ң нҢҢмқјм—җ лӢҙкі  мӢ¶лӢӨл©ҙ м •м Ғ л©ӨлІ„ нҒҙлһҳмҠӨ(м•„мқҙн…ң 24)лҘј мӮ¬мҡ©н•ҳлҠ” л°©лІ•мқ„ кі лҜјн•ҙліј мҲҳ мһҲлӢӨ.
    • мқҙлҠ” мқҪкё° мўӢкі , privateмңјлЎң м„ м–ён•ҳл©ҙ(м•„мқҙн…ң 15) м ‘к·ј лІ”мң„лҸ„ мөңмҶҢлЎң кҙҖлҰ¬н•  мҲҳ мһҲкё° л•Ңл¬ёмқҙлӢӨ.
