Arrayмқҳ к°ңл…җ

  • к°ҷмқҖ нғҖмһ…мқҳ м—¬лҹ¬ лҚ°мқҙн„°лҘј н•ҳлӮҳмқҳ мқҙлҰ„мңјлЎң к·ёлЈ№н•‘н•ҙм„ң кҙҖлҰ¬н•ҳкё° мң„н•ң мһҗлЈҢкө¬мЎ°
  • indexмҷҖ к°’мқҳ мҢҚмңјлЎң кө¬м„ұ
  • indexлҠ” к°’м—җ лҢҖн•ң мң мқјл¬ҙмқҙн•ң мӢқлі„мһҗ(Listм—җм„ң мқёлҚұмҠӨлҠ” лӘҮ лІҲм§ё лҚ°мқҙн„°мқёк°Җ м •лҸ„мқҳ мқҳлҜёлҘј к°Җм§җ)
  • м—°мҶҚлҗң л©”лӘЁлҰ¬мқҳ кіөк°„мңјлЎң мқҙлЈЁм–ҙм ё мһҲлӢӨ.
  • л…јлҰ¬м Ғ м ҖмһҘ мҲңм„ңмҷҖ л¬јлҰ¬м Ғ м ҖмһҘ мҲңм„ңк°Җ мқјм№ҳ => indexлЎң н•ҙлӢ№ мӣҗмҶҢм—җ м ‘к·јн•  мҲҳ мһҲлӢӨ. (мӢңк°„ліөмһЎлҸ„: O(1))
  • л°°м—ҙмқҖ м„ м–ёкіј лҸҷмӢңм—җ кёёмқҙлҘј м§Җм •н•ҳл©° кёёмқҙлҘј ліҖкІҪн•  мҲҳ м—ҶлӢӨ.

Arrayмқҳ мһҘм җ

  • мқёлҚұмҠӨлҘј нҶөн•ң кІҖмғүмқҙ мҡ©мқҙн•Ё
  • м—°мҶҚм ҒмқҙлҜҖлЎң л©”лӘЁлҰ¬ кҙҖлҰ¬к°Җ нҺён•Ё

Arrayмқҳ лӢЁм җ

  • л°°м—ҙмқҳ нҒ¬кё°лҘј м»ҙнҢҢмқјн•ҳкё°м „м—җ м •н•ҙмЈјм–ҙм•ј н•ңлӢӨ.
  • нҒ¬кё°к°Җ кі м •лҗҳм–ҙ мһҲм–ҙм„ң м»ҙнҢҢмқј мқҙнӣ„ л°°м—ҙмқҳ нҒ¬кё°лҘј ліҖкІҪн•  мҲҳ м—ҶлӢӨ.
  • нҒ¬кё°к°Җ кі м •лҗҳм–ҙ мһҲкё° л•Ңл¬ём—җ м–ҙл–Ө м—ҳлҰ¬лЁјнҠёк°Җ мӮӯм ңлҗҳл©ҙ, мӮӯм ңлҗң мғҒнғңлҘј л№Ҳ кіөк°„мңјлЎң лӮЁкІЁл‘җм–ҙм•ј н•ҳкё°м—җ л©”лӘЁлҰ¬ лӮӯл№„к°Җ л°ңмғқн•ңлӢӨ.

Listмқҳ к°ңл…җ

  • мҲңм„ңк°Җ мһҲлҠ” лҚ°мқҙн„°л“Өмқҳ м§‘н•©мңјлЎң л°°м—ҙкіјлҠ” лӢӨлҘҙкІҢ л№Ҳ м—ҳлҰ¬лЁјнҠёлҠ” м ҲлҢҖ н—Ҳмҡ©н•ҳм§Җ м•ҠлҠ”лӢӨ.
  • л°°м—ҙмқҙ к°Җм§„ мқёлҚұмҠӨлқјлҠ” мһҘм җмқ„ лІ„лҰ¬кі  лҢҖмӢ  л№ҲнӢҲм—ҶлҠ” лҚ°мқҙн„°мқҳ м Ғмһ¬лқјлҠ” мһҘм җмқҙ мһҲлӢӨ.
  • Listм—җм„ңмқҳ мқёлҚұмҠӨлҠ” лӘҮ лІҲм§ё лҚ°мқҙн„°мқёк°Җ м •лҸ„(мҲңм„ң)мқҳ мқҳлҜёлҘј к°Җм§„лӢӨ(л°°м—ҙм—җм„ңмқҳ мқёлҚұмҠӨлҠ” к°’м—җ лҢҖн•ң мң мқјл¬ҙмқҙн•ң мӢқлі„мһҗ)
  • л№Ҳ м—ҳлҰ¬лЁјнҠёлҠ” н—Ҳмҡ©н•ҳм§Җ м•ҠлҠ”лӢӨ. => javaм—җм„ңлҠ” н—Ҳмҡ©н•ҳлҠ” кІҪмҡ°лҸ„ мһҲмқҢ
  • мҲңм°Ём„ұмқ„ ліҙмһҘн•ҳм§Җ лӘ»н•ҳкё° л•Ңл¬ём—җ spacial locality ліҙмһҘмқҙ лҗҳм§Җ м•Ҡм•„м„ң cash hitк°Җ м–ҙл өлӢӨ.(лҚ°мқҙн„° к°ҜмҲҳк°Җ нҷ•мӢӨн•ҳкІҢ м •н•ҙм ё мһҲкі , мһҗмЈј мӮ¬мҡ©лҗңлӢӨл©ҙ arrayк°Җ лҚ” нҡЁмңЁм ҒмқҙлӢӨ.)
  • л¶Ҳм—°мҶҚм ҒмңјлЎң л©”лӘЁлҰ¬ кіөк°„мқ„ м°Ём§Җ
  • нҸ¬мқён„°лҘј нҶөн•ң м ‘к·ј

Listмқҳ мһҘм җ

  • нҸ¬мқён„°лҘј нҶөн•ҳм—¬ лӢӨмқҢ лҚ°мқҙн„°мқҳ мң„м№ҳлҘј к°ҖлҘҙнӮӨкі  мһҲкё°м—җ мӮҪмһ… мӮӯм ңк°Җ мҡ©мқҙн•ҳлӢӨ.
  • л©”лӘЁлҰ¬ кіөк°„мқҳ нҒ¬кё°лҘј лҸҷм ҒмңјлЎң ліҖкІҪн•  мҲҳ мһҲлӢӨ.(м»ҙнҢҢмқј нӣ„ лҚ°мқҙн„° м¶”к°Җ к°ҖлҠҘ)
  • л©”лӘЁлҰ¬мқҳ мһ¬мӮ¬мҡ©мқҙ нҺёлҰ¬н•ҳлӢӨ.
  • л¶Ҳм—°мҶҚм ҒмқҙлҜҖлЎң л©”лӘЁлҰ¬ кҙҖлҰ¬к°Җ нҺёлҰ¬н•ҳлӢӨ.

Listмқҳ лӢЁм җ

  • кІҖмғү м„ұлҠҘмқҙ мўӢм§Җ м•ҠлӢӨ.
  • нҸ¬мқён„°лҘј нҶөн•ҙ лӢӨмқҢ лҚ°мқҙн„°лҘј к°ҖлҰ¬нӮӨлҜҖлЎң м¶”к°Җм Ғмқё л©”лӘЁлҰ¬ кіөк°„ л°ңмғқ.

ArrayмҷҖ Listмқҳ мӮҪмһ…, мӮӯм ң, мЎ°нҡҢ л№„көҗ

||м¶”к°Җ/мӮӯм ң|мЎ°нҡҢ| |вҖ”вҖ“|вҖ”вҖ“|вҖ”вҖ“| |Array|лҠҗлҰј|л№ лҰ„| |List|л№ лҰ„|лҠҗлҰј|

  • л°°м—ҙ: лҚ°мқҙн„°мқҳ нҒ¬кё°к°Җ м •н•ҙм ё мһҲкі , м¶”к°Җм Ғмқё мӮҪмһ… мӮӯм ңк°Җ мқјм–ҙлӮҳм§Җ м•Ҡмңјл©° кІҖмғүмқ„ н•„мҡ”лЎң н•  л•Ң мң лҰ¬н•ҳлӢӨ.
  • лҰ¬мҠӨнҠё: лҚ°мқҙн„°мқҳ нҒ¬кё°к°Җ к°ҖліҖм Ғмқҙкі , мӮҪмһ… мӮӯм ңк°Җ л§Һмқҙ мқјм–ҙлӮҳл©°, кІҖмғүмқҙ м ҒмқҖ кІҪмҡ°м—җ мң лҰ¬н•ҳлӢӨ.

Java List Collection

  • ListлҠ” Collection мқён„°нҺҳмқҙмҠӨлҘј нҷ•мһҘн•ң мһҗлЈҢнҳ•мңјлЎң лҸҷмқјн•ң лҚ°мқҙн„°мқҳ мӨ‘ліө мһ…л Ҙмқҙ к°ҖлҠҘн•ҳлӢӨ.
  • мЈјлЎң мҲңм°Ём Ғмқҙкі  лӢӨлҹүмқҳ лҚ°мқҙн„°лҘј мһ…л Ҙн•  л•Ң мӮ¬мҡ©н•ңлӢӨ.
  • мў…лҘҳлҠ” Vector, Arraylist, LinkedListк°Җ мһҲлӢӨ.

ArrayList

  • мқјл°ҳ л°°м—ҙкіј ArrayListлҠ” мқёлҚұмҠӨлЎң к°қмІҙлҘј кҙҖлҰ¬н•ңлӢӨлҠ” м җмқҖ лҸҷмқјн•ҳм§Җл§Ң, ArrayListлҠ” нҒ¬кё°лҘј лҸҷм ҒмңјлЎң лҠҳлҰҙ мҲҳ мһҲлӢӨлҠ” м җм—җм„ң м°Ёмқҙм җмқҙ мЎҙмһ¬н•ңлӢӨ.
  • ArrayListлҠ” лӮҙл¶Җм—җм„ң мІҳмқҢ м„Өм •н•ң м ҖмһҘ мҡ©лҹү(capacity)к°Җ мһҲлӢӨ.(DEFAULT_CAPACITY=10)
  • лҚ°мқҙн„° м¶”к°Җ мӢң кіөк°„мқҙ л¶ҖмЎұн•ң кІҪмҡ° л°°м—ҙ нҒ¬кё°лҘј 1.5л°°лЎң мҰқк°ҖмӢңнӮЁлӢӨ.

  • ArrayListмҷҖ LinkedListмқҳ м°Ёмқҙ

л°°м—ҙкіј ArrayListмқҳ м°Ёмқҙ

  • л°°м—ҙмқҖ нҒ¬кё°к°Җ кі м •лҗҳм–ҙмһҲм§Җл§Ң ArrayListлҠ” мӮ¬мқҙмҰҲк°Җ лҸҷм Ғмқё л°°м—ҙмқҙлӢӨ.
  • л°°м—ҙмқҖ primitive type(int, byte, char л“ұ)кіј object лӘЁл‘җлҘј лӢҙмқ„ мҲҳ мһҲм§Җл§Ң, ArrayListлҠ” object elementл§Ң лӢҙмқ„ мҲҳ мһҲлӢӨ.
  • л°°м—ҙмқҖ м ңл„ӨлҰӯмқ„ мӮ¬мҡ©н•  мҲҳ м—Ҷм§Җл§Ң, ArrayListлҠ” нғҖмһ… м•Ҳм •м„ұмқ„ ліҙмһҘн•ҙмЈјлҠ” м ңл„ӨлҰӯмқ„ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.
  • кёёмқҙм—җ лҢҖн•ҙ л°°м—ҙмқҖ length ліҖмҲҳлҘј м“°кі , arrayListлҠ” size() л©”м„ңл“ңлҘј мҚЁм•јн•ңлӢӨ.
  • л°°м—ҙмқҖ elementл“Өмқ„ н• лӢ№н•ҳкё° мң„н•ҙ assignment(н• лӢ№) м—°мӮ°мһҗлҘј мҚЁм•јн•ҳкі , ArrayListлҠ” add() л©”м„ңл“ңлҘј нҶөн•ҙ elementлҘј мӮҪмһ…н•ңлӢӨ.

private Object[] grow(int minCapacity) {
        // newCapacity кёёмқҙмқҳ мғҲ л°°м—ҙм—җ кё°мЎҙ л°°м—ҙ ліөмӮ¬
        return this.elementData = Arrays.copyOf(this.elementData, this.newCapacity(minCapacity));
    }
    
 private int newCapacity(int minCapacity) {
        int oldCapacity = this.elementData.length;
        // кё°мЎҙ лҰ¬мҠӨнҠё кёёмқҙ + (кё°мЎҙлҰ¬мҠӨнҠёкёёмқҙ/2)
        // кёёмқҙк°Җ л¶ҖмЎұн•  кІҪмҡ° 1.5л°° кёёмқҙлҘј лҠҳлҰ°лӢӨ
        int newCapacity = oldCapacity + (oldCapacity >> 1);
        if (newCapacity - minCapacity <= 0) {
            if (this.elementData == DEFAULTCAPACITY_EMPTY_ELEMENTDATA) {
                return Math.max(10, minCapacity);
            } else if (minCapacity < 0) {
                throw new OutOfMemoryError();
            } else {
                return minCapacity;
            }
        } else {
            return newCapacity - 2147483639 <= 0 ? newCapacity : hugeCapacity(minCapacity);
        }
    }

ArrayListмқҳ мӮҪмһ…/мӮӯм ң кіјм •

LinkedList

  • л…ёл“ң к°„м—җ м—°кІ°(link)мқ„ нҶөн•ҙм„ң лҰ¬мҠӨнҠёлЎң кө¬нҳ„лҗң к°қмІҙмқҙлӢӨ.
  • лӢӨмқҢ л…ёл“ңмқҳ мң„м№ҳ м •ліҙл§Ң к°Җм§Җкі  мһҲмңјл©° мқёлҚұмҠӨлҘј к°Җм§Җкі  мһҲм§Җ м•Ҡкё° л•Ңл¬ём—җ нғҗмғүмӢң мҲңм°Ём ‘к·јл§Ң к°ҖлҠҘн•ҳлӢӨ(randomAccess л¶Ҳк°ҖлҠҘ л°Ҹ нғҗмғү мӢң мӢңк°„мқҙ л§Һмқҙ мҶҢмҡ”лҗ  мҲҳ мһҲмқҢ)
  • л…ёл“ң мӮҪмһ…/мӮӯм ңлҠ” мң„м№ҳ м •ліҙмқҳ мҲҳм •л§ҢмңјлЎң к°ҖлҠҘн•ҳкё° л•Ңл¬ём—җ м„ұлҠҘмқҙ мўӢлӢӨ.
  • LinkedListлҠ” ArrayListмҷҖ лӢ¬лҰ¬ Listмқён„°нҺҳмқҙмҠӨлҘј кө¬нҳ„н•ң AbstractListлҘј мғҒмҶҚн•ҳм§Җ м•Ҡкі  AbstractSquentialListлҘј мғҒмҶҚн•ңлӢӨ.

LinkedListмқҳ мӮҪмһ…/мӮӯм ң кіјм •


Vector

  • VectorлҠ” ArrayListмҷҖ лҸҷмқјн•ң лӮҙл¶Җ кө¬мЎ°лҘј к°Җм§Җкі  мһҲлӢӨ. Vector к°қмІҙлҘј мғқм„ұн•ҳкё° мң„н•ҙм„ңлҠ” м ҖмһҘн•  нғҖмһ…мқ„ м§Җм •н•ҙм•ј н•ңлӢӨ.
  • ArrayListмҷҖ м°Ёмқҙм җмңјлЎңлҠ” Vector нҒҙлһҳмҠӨлҠ” лҸҷкё°нҷ”лҗң(synchronized) л©”м„ңл“ңлЎң кө¬м„ұлҗҳм–ҙ мһҲлӢӨ.
    • л”°лқјм„ң л©ҖнӢ° мҠӨл Ҳл“ң нҷҳкІҪм—җм„ң м•Ҳм „н•ҳкІҢ к°қмІҙлҘј мӮҪмһ…/мӮӯм ңн•  мҲҳ мһҲлӢӨ.(Thread-Safe)
  • н•ҳм§Җл§Ң Thread-safeн•ҳкё° л•Ңл¬ём—җ lockмқ„ кұёкі  н‘ёлҠ” кіјм •мқ„ л°ҳл“ңмӢң кұ°міҗм•јн•ҳлҜҖлЎң мӮҪмһ…/мӮӯм ңмӢң лӢӨмҶҢ лҠҗлҰ¬лӢӨ.
  • лҚ°мқҙн„° м¶”к°Җ мӢң кіөк°„мқҙ л¶ҖмЎұн•ң кІҪмҡ° л°°м—ҙ нҒ¬кё°лҘј 2л°°лЎң мҰқк°ҖмӢңнӮЁлӢӨ.
