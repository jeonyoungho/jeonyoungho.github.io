Kafka 기본 실습

jeonyoungho Jun 30, 2021 2021-06-30T00:00:00+09:00
1 min

Kafka-실습1

Kafka 설치

1) homebrew를 통한 kafka 설치

1

brew install kafka

Kafka 실행

2) Kafka를 실행하기 전 zookeeper를 먼저 시작

1

brew services start zookeeper


3) Kafka 시작

1

brew services start kafka


4) Kafka에서 기본적으로 제공하는 테스트를 위한 다양한 script의 위치 확인

1

brew info kafka



5) Topic생성 (파티션1개 replication은 1이고 Topic이름은 test)

1

./kafka-topics --create --zookeeper localhost:2181 --replication-factor 1 -partitions 1 --topic test


6) 로컬에 띄워진 카프카에 생성한 Topic으로 String데이터를 전송

1

./kafka-console-producer --broker-list localhost:9092 --topic test


7) Topic으로 부터 데이터를 맨 처음 offset부터 읽어오기

1

./kafka-console-consumer --bootstrap-server localhost:9092 --topic test --from-beginning
